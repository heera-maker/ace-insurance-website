ACE INSURANCE INC — 3D WEBSITE

FILES
- index.html: website page
- styles.css: design and responsive layout
- script.js: 3D movement, menu and quote email form
- assets/ace-logo.png: exact crop of the logo from your supplied image; it was not redrawn
- assets/heera-singh.png: founder photo crop from your supplied image
- assets/ace-flyer-original.png: original supplied promotional image

PREVIEW
Open index.html in Chrome, Edge or Safari.

LAUNCH ON NETLIFY (EASIEST)
1. Go to https://app.netlify.com/drop
2. Drag the entire ace-3d-website folder onto the page.
3. Netlify publishes it and gives you a temporary link.
4. In Netlify, open Domain management > Add a domain > Add a domain you already own.
5. Enter aceins.co and follow the DNS records Netlify gives you.

LAUNCH ON GITHUB PAGES
1. Create a new public GitHub repository named ace-3d-website.
2. Upload all files and the assets folder.
3. Open Settings > Pages.
4. Under Build and deployment choose Deploy from a branch.
5. Select main and /root, then Save.
6. GitHub will give you a live website link.

IMPORTANT
The quote form opens the visitor's email app and prepares an email to heera@aceins.co.
Confirm the Facebook page URL. The current link is https://www.facebook.com/aceinsuranceinc.
